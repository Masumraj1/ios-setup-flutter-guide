# Qorum App (iOS Setup Guide)

এই প্রজেক্টটি একটি Flutter অ্যাপ্লিকেশন যা iOS এবং Android উভয় প্ল্যাটফর্মে কাজ করে।

## iOS কনফিগারেশন এবং Pod সেটিংস

আপনার iOS ডিভাইসে অ্যাপটি সঠিকভাবে রান করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

### ১. Deployment Target পরিবর্তন
অ্যাপটি সর্বনিম্ন **iOS 14.0** ভার্সন সাপোর্ট করে। এটি পরিবর্তন করতে:
* `ios/Podfile` ফাইলে গিয়ে `platform :ios, '14.0'` লাইনটি নিশ্চিত করুন।

### ২. CocoaPods রিসেট এবং ইন্সটল
যদি iOS বিল্ডে কোনো এরর দেখা দেয়, তবে টার্মিনালে নিচের কমান্ডগুলো সিরিয়ালি রান করুন:

```bash
# ios ফোল্ডারে প্রবেশ করুন
cd ios

# পুরনো Pods এবং Lock ফাইল মুছে ফেলুন
rm -rf Pods
rm Podfile.lock

# নতুন করে লাইব্রেরিগুলো ইন্সটল করুন
pod install
